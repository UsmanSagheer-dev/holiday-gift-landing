### Change Product Name
Find and replace "Panda Drum" with your product name in `index.html`

### Change Colors
Update colors in the Tailwind config section of `index.html`:
```javascript

    brandPink: "#YOUR_COLOR",     // Main color
    brandLight: "#YOUR_COLOR2",   // Accent color
}
```

Replace files in `/assets/images/`:
- `reason1Image.png` - Main product image
- `shopNow.png` - Offer showcase image  
- `Vector.png` - Checkmark icons
- `star.svg` - Rating stars
